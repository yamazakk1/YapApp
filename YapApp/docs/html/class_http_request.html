Detailed Description

Класс для представления и обработки HTTP-запросов

Содержит разобранные компоненты HTTP-запроса:

  • Метод (GET/POST/etc)
  • Путь
  • Заголовки
  • Query-параметры
  • Тело запроса
  • Статус валидности
